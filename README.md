# Life Simulation Engine (Text-Based Simulator)

## Overview

The Life Simulation Engine is a console-based application developed in C++ using Object-Oriented Programming (OOP) principles. The project simulates a virtual human life where users make daily decisions that influence their character's future. Through actions such as studying, working, resting, and exercising, players experience how small choices can lead to long-term consequences.

## Features

* Interactive text-based gameplay
* Multiple character attributes:

  * Health
  * Energy
  * Money
  * Skill
  * Happiness
* Daily decision-making system
* Random life events and opportunities
* Final performance evaluation
* Modular OOP design

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Console-based Interface

## OOP Concepts Implemented

* Classes and Objects
* Encapsulation
* Abstraction
* Inheritance
* Modularity

## How It Works

The simulation progresses through multiple virtual days. Each day, the player chooses an activity that affects the character's attributes. The system continuously updates these values while maintaining logical limits. Random events may occur, adding realism and unpredictability to the simulation. At the end, a detailed report evaluates the player's life decisions and overall performance.
